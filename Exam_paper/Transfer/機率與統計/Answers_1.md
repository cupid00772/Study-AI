## 一、計算與證明題：聯合機率分佈 (40分)

### 1. 邊際機率質量函數 $P_X(x)$ 與 $P_Y(y)$ [Easy]
**解題步驟**：
將 Joint PMF 表格的各列 (Row) 與各行 (Column) 加總即可得到邊際機率。
*   $P_X(0) = 0.10 + 0.05 + 0.05 = 0.20$
*   $P_X(1) = 0.20 + 0.15 + 0.10 = 0.45$
*   $P_X(2) = 0.10 + 0.15 + 0.10 = 0.35$

*   $P_Y(0) = 0.10 + 0.20 + 0.10 = 0.40$
*   $P_Y(1) = 0.05 + 0.15 + 0.15 = 0.35$
*   $P_Y(2) = 0.05 + 0.10 + 0.10 = 0.25$

> [!WARNING] 常見錯誤陷阱
> 容易把行與列加總顛倒，請記住 $P_X(x)$ 是固定 $X$ 加總所有的 $Y$。

### 2. 證明獨立性 [Medium]
**解題步驟**：
*   **核心定理**：若 $X$ 和 $Y$ 獨立，則必須滿足對**所有**的 $(x, y)$，都有 $P_{X,Y}(x, y) = P_X(x) \times P_Y(y)$。
*   檢查 $(x=0, y=0)$ 的情況：
    $P_X(0) \times P_Y(0) = 0.20 \times 0.40 = 0.08$
*   然而由表可知 $P_{X,Y}(0, 0) = 0.10$。
*   因為 $0.08 \neq 0.10$，**故 $X$ 與 $Y$ 不獨立**。

### 3. 計算 $E[X]$ 與 $E[Y]$ [Easy]
**解題步驟**：
利用邊際機率與期望值公式 $E[X] = \sum x P_X(x)$。
*   $E[X] = 0(0.20) + 1(0.45) + 2(0.35) = 0.45 + 0.70 = 1.15$
*   $E[Y] = 0(0.40) + 1(0.35) + 2(0.25) = 0.35 + 0.50 = 0.85$

### 4. 計算 $E[X+Y]$ 與 $E[XY]$ [Medium]
**解題步驟**：
*   由期望值的線性性質：$E[X+Y] = E[X] + E[Y] = 1.15 + 0.85 = 2.0$。
*   利用 $E[XY] = \sum_x \sum_y (xy) P_{X,Y}(x, y)$。只要考慮 $x, y$ 皆不為 0 的項：
    $E[XY] = (1\times1)(0.15) + (1\times2)(0.10) + (2\times1)(0.15) + (2\times2)(0.10)$
    $E[XY] = 0.15 + 0.20 + 0.30 + 0.40 = 1.05$

## 二、應用題：二項式與幾何分佈綜合 (35分)

### 1. Binomial Distribution 分佈與機率 [Easy]
**解題步驟**：
*   **分佈名稱**：二項式分佈 (Binomial Distribution)
*   **參數**：$W \sim Bin(15, 0.1)$，其中 $n = 15$, $p = 0.1$。
*   **機率計算**：$P(W = 2) = \binom{15}{2} (0.1)^2 (0.9)^{13}$

### 2. 期望值與變異數 [Easy]
**核心公式**：對於 $Bin(n, p)$，$E[W] = np$, $Var(W) = np(1-p)$。
*   $E[W] = 15 \times 0.1 = 1.5$
*   $Var(W) = 15 \times 0.1 \times 0.9 = 1.35$

### 3. 幾何分佈機率計算 [Medium]
**解題步驟**：
*   **分佈名稱**：幾何分佈 (Geometric Distribution)
*   **機率計算**：因為是在「第 5 次」才首度發現，表示前 4 次都是良品，第 5 次是不良品。
    $P(Z = 5) = (0.9)^4 (0.1)$

> [!WARNING] 常見錯誤陷阱
> 計算幾何分佈時，不要再乘上組合數 $C(5,1)$，因為發生的順序是被固定好的（前四次失敗、第五次成功）。

## 三、應用題：卜瓦松分佈與近似 (35分)

### 1. 卜瓦松分佈 PMF [Easy]
**解題步驟**：
*   $N \sim Poisson(\lambda)$，平均每小時 $\lambda = 3$。
*   **PMF**: $P(N = k) = \frac{e^{-3} 3^k}{k!}$
*   代入 $k = 4$: $P(N = 4) = \frac{e^{-3} 3^4}{4!} = \frac{81 e^{-3}}{24} = \frac{27}{8} e^{-3}$

### 2. 時間區間調整與「至少」的機率 [Medium]
**解題步驟**：
*   時間改為「兩小時」，因此新的期望值 $\lambda' = 3 \times 2 = 6$。令 $N_2 \sim Poisson(6)$。
*   求「至少一次」應使用補集概念：$P(N_2 \ge 1) = 1 - P(N_2 = 0)$。
*   $P(N_2 \ge 1) = 1 - \frac{e^{-6} 6^0}{0!} = 1 - e^{-6}$

> [!WARNING] 常見錯誤陷阱
> 忘記將 $\lambda$ 根據時間區間的改變做等比例的放大（這裡是乘以 2）。

### 3. 卜瓦松近似二項式 [Hard]
**解題步驟**：
*   原始問題為 $X \sim Bin(1000, 0.002)$，要求 $P(X=3)$。
*   因為 $n$ 很大且 $p$ 很小，可以使用 Poisson 分佈作為近似。
*   **核心公式**：設定 Poisson 的參數 $\lambda = n \times p = 1000 \times 0.002 = 2$。
*   **近似結果**：$P(X=3) \approx \frac{e^{-2} 2^3}{3!} = \frac{8 e^{-2}}{6} = \frac{4}{3} e^{-2}$
