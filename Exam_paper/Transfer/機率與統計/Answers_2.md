## 一、基礎代入題：二項式與幾何分佈 (40分)

### 1. 二項式分佈公式 [Easy]
**解題步驟**：
*   **分佈名稱**：二項式分佈 (Binomial Distribution)
*   **核心公式**：$P(X = x) = \binom{n}{x} p^x (1-p)^{n-x}$
*   **代入參數**：已知 $n=5, p=0.2$，故 $P(X = x) = \binom{5}{x} (0.2)^x (0.8)^{5-x}$。

### 2. 二項式機率計算 [Easy]
**解題步驟**：
*   要求 $X = 2$ (恰好 2 個瑕疵品)，直接將 $x=2$ 代入剛剛的公式。
*   $P(X = 2) = \binom{5}{2} (0.2)^2 (0.8)^3$
    $= 10 \times 0.04 \times 0.512$
    $= 0.2048$
*(註：只要寫出 $\binom{5}{2} (0.2)^2 (0.8)^3$ 即可拿全部分數)*

### 3. 幾何分佈判斷 [Easy]
**解題步驟**：
*   **分佈名稱**：幾何分佈 (Geometric Distribution)。
*   **判斷關鍵**：題目明確指出「不斷重複，**直到他第一次抽到**...」，這就是幾何分佈最標準的特徵。

### 4. 幾何機率計算 [Easy]
**解題步驟**：
*   **核心公式**：$P(Y = y) = (1-p)^{y-1} \times p$
*   要求 $Y = 3$ (第 3 次才中獎，代表前 2 次都沒中，第 3 次中)。
*   $P(Y = 3) = (1 - 0.2)^{3-1} \times 0.2 = (0.8)^2 \times 0.2 = 0.64 \times 0.2 = 0.128$

> [!WARNING] 常見錯誤陷阱
> 幾何分佈的公式前面**絕對不會有**組合數 $\binom{n}{x}$！因為順序已經固定了（失敗、失敗、成功）。

## 二、基礎代入題：卜瓦松分佈 (35分)

### 1. 卜瓦松 PMF 公式 [Easy]
**解題步驟**：
*   已知每小時平均發生率 $\lambda = 4$。
*   **核心公式**：$P(N = k) = \frac{e^{-\lambda} \lambda^k}{k!}$
*   **代入參數**：$P(N = k) = \frac{e^{-4} 4^k}{k!}$

### 2. 基本機率計算 [Easy]
**解題步驟**：
*   要求 1 小時內恰好 3 名病患，即求 $P(N = 3)$。
*   $P(N = 3) = \frac{e^{-4} 4^3}{3!} = \frac{64 e^{-4}}{6} = \frac{32}{3} e^{-4}$

### 3. 區間調整與計算 [Easy]
**解題步驟**：
*   **核心觀念**：時間區間從 1 小時變成 2 小時，發生的平均次數也要乘以 2。
*   **新的 $\lambda$**：$\lambda' = 4 \times 2 = 8$
*   求恰好 5 名，代入新參數的卜瓦松公式：$P(N = 5) = \frac{e^{-8} 8^5}{5!}$

> [!WARNING] 常見錯誤陷阱
> 考試時只要看到「單位時間、長度、面積」改變，第一步就是要重算你的 $\lambda$，切勿直接拿原本的數字硬代。

## 三、基礎代入題：聯合機率分佈 Joint PMF (35分)

### 1. 計算 $X$ 的邊際機率 [Easy]
**解題步驟**：
*   要把 $X$ 算出來，就是把包含它的「整列」加起來。
*   $P_X(0) = P(0,0) + P(0,1) = 0.2 + 0.4 = 0.6$
*   $P_X(1) = P(1,0) + P(1,1) = 0.3 + 0.1 = 0.4$

### 2. 計算 $Y$ 的邊際機率 [Easy]
**解題步驟**：
*   要把 $Y$ 算出來，就是把包含它的「整行」加起來。
*   $P_Y(0) = P(0,0) + P(1,0) = 0.2 + 0.3 = 0.5$
*   $P_Y(1) = P(0,1) + P(1,1) = 0.4 + 0.1 = 0.5$

### 3. 檢驗獨立性 [Easy]
**解題步驟**：
*   **核心檢驗式**：$X$ 和 $Y$ 獨立，必須滿足 $P(X=0, Y=0) = P_X(0) \times P_Y(0)$。
*   從前面算出的邊際機率得知：$P_X(0) \times P_Y(0) = 0.6 \times 0.5 = 0.3$。
*   回到原本的表格查找：$P(X=0, Y=0) = 0.2$。
*   因為 $0.3 \neq 0.2$，左邊不等於右邊，**所以 $X$ 與 $Y$ 不獨立 (Not Independent)**。
