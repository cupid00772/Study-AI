## 一、聯合機率質量函數與條件機率 (40分)

### 1. 求常數 $c$ [Medium]
**解題步驟**：
*   **核心觀念**：所有可能的聯合機率總和必須等於 1，即 $\sum_x \sum_y P_{X,Y}(x,y) = 1$。
*   列出所有可能的組合：
    $P(1,1) = 2c, P(1,2) = 3c$
    $P(2,1) = 3c, P(2,2) = 4c$
    $P(3,1) = 4c, P(3,2) = 5c$
*   總和 = $2c + 3c + 3c + 4c + 4c + 5c = 21c$
*   故 $21c = 1 \implies c = \frac{1}{21}$。

### 2. 邊際機率質量函數 [Medium]
**解題步驟**：
*   $P_X(x) = \sum_y P(x,y) = P(x,1) + P(x,2) = c(x+1) + c(x+2) = c(2x+3) = \frac{2x+3}{21}$, 對於 $x=1,2,3$。
*   $P_Y(y) = \sum_x P(x,y) = c(1+y) + c(2+y) + c(3+y) = c(3y+6) = \frac{3y+6}{21} = \frac{y+2}{7}$, 對於 $y=1,2$。

### 3. 條件機率 [Hard]
**解題步驟**：
*   **核心公式**：$P(X \ge 2 \mid Y = 1) = \frac{P(X \ge 2 \text{ 且 } Y = 1)}{P(Y=1)}$
*   分子 $P(X \ge 2, Y = 1) = P(2,1) + P(3,1) = 3c + 4c = 7c = \frac{7}{21} = \frac{1}{3}$。
*   分母 $P_Y(1) = \frac{1+2}{7} = \frac{3}{7}$ (或寫 $\frac{9}{21}$)。
*   結果：$\frac{7/21}{9/21} = \frac{7}{9}$。

### 4. 獨立性檢驗 [Medium]
**解題步驟**：
*   檢驗 $P(1,1)$ 是否等於 $P_X(1) \times P_Y(1)$。
*   $P(1,1) = 2c = \frac{2}{21} = \frac{42}{441}$。
*   $P_X(1) = \frac{5}{21}$，$P_Y(1) = \frac{9}{21}$，兩者相乘 = $\frac{45}{441}$。
*   因為 $\frac{42}{441} \neq \frac{45}{441}$，故 $X$ 與 $Y$ **不獨立 (Not Independent)**。

## 二、數位通訊系統分析 (35分)

### 1. 第一次錯誤的機率 (Geometric) [Easy]
**解題步驟**：
*   **判斷**：直到第 1000 個位元才「第一次」發生錯誤 $\rightarrow$ 幾何分佈。
*   $P(Y = 1000) = (1-p)^{999} p = (1 - 10^{-3})^{999} (10^{-3}) = (0.999)^{999} \times 0.001$。

### 2. 封包被丟棄的機率 (Binomial) [Medium]
**解題步驟**：
*   **判斷**：在固定的 1000 個位元中，求錯誤發生的次數 $\rightarrow$ 二項式分佈。
*   令 $X \sim Bin(1000, 10^{-3})$。丟棄條件為 $X \ge 2$。
*   使用補集：$P(X \ge 2) = 1 - P(X = 0) - P(X = 1)$。
*   $P(X = 0) = \binom{1000}{0} p^0 (1-p)^{1000} = (0.999)^{1000}$。
*   $P(X = 1) = \binom{1000}{1} p^1 (1-p)^{999} = 1000 \times 0.001 \times (0.999)^{999} = (0.999)^{999}$。
*   **答案**：$1 - (0.999)^{1000} - (0.999)^{999}$。

### 3. 複合式期望值 (Expected Value of Binomial) [Hard]
**解題步驟**：
*   這題是雙重概念：先找出「單一封包恰好 1 個錯誤」的機率，再求 50 個封包的期望值。
*   單一封包恰好 1 個錯誤的機率 $P(X=1) = (0.999)^{999}$。(上題已算過)
*   對於這 50 個封包，符合條件的封包數量 $W \sim Bin(50, P(X=1))$。
*   預期數量 (期望值) $E[W] = n \times P(X=1) = 50 \times (0.999)^{999}$。

## 三、伺服器負載與機率近似 (35分)

### 1. 區間調整：10 分鐘 [Medium]
**解題步驟**：
*   原始 $\lambda = 180$ 次 / 60 分鐘。
*   10 分鐘的平均次數 $\lambda_{10} = 180 \times \frac{10}{60} = 30$。
*   代入 Poisson 公式：$P(N = 25) = \frac{e^{-30} 30^{25}}{25!}$。

### 2. 區間調整與過載機率 [Medium]
**解題步驟**：
*   1 分鐘的平均次數 $\lambda_1 = 180 \times \frac{1}{60} = 3$。
*   過載條件：收到超過 4 次 (即 $X > 4$)。
*   $P(X > 4) = 1 - [P(X=0) + P(X=1) + P(X=2) + P(X=3) + P(X=4)]$
*   $= 1 - e^{-3} [\frac{3^0}{0!} + \frac{3^1}{1!} + \frac{3^2}{2!} + \frac{3^3}{3!} + \frac{3^4}{4!}]$
*   $= 1 - e^{-3} [1 + 3 + 4.5 + 4.5 + 3.375] = 1 - 16.375 e^{-3}$。

### 3. Poisson 近似 Binomial [Medium]
**解題步驟**：
*   **判斷**：$n = 20000$ (極大)，$p = 0.0002$ (極小)，符合 Poisson 近似的條件。
*   **核心公式**：近似的 $\lambda = np = 20000 \times 0.0002 = 4$。
*   以 $Poisson(4)$ 計算恰好 5 個故障的機率：$P(X=5) \approx \frac{e^{-4} 4^5}{5!} = \frac{1024 e^{-4}}{120} = \frac{128}{15} e^{-4}$。
