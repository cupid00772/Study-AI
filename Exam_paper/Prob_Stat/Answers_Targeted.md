## 詳細解答與步驟解析

### 🎯 弱點打擊 1：獨立與互斥 (Ch 2)

**1. 基本定義再確認**
- **(a) 獨立事件：**
  - **步驟：** 獨立代表交集機率相乘。$P(C \cap D) = 0.3 \times 0.6 = 0.18$。
  - **計算：** $P(C \cup D) = P(C) + P(D) - P(C \cap D) = 0.3 + 0.6 - 0.18 = 0.72$。
- **(b) 互斥事件：**
  - **步驟：** 互斥代表交集機率為 0。$P(C \cap D) = 0$。
  - **計算：** $P(C \cup D) = 0.3 + 0.6 - 0 = 0.9$。

**2. 條件機率的邏輯**
- **事件定義：** $A = \{2, 4, 6\}$，$B = \{4, 5, 6\}$。
- $A \cup B = \{2, 4, 5, 6\}$，共有 4 個結果，機率 $P(A \cup B) = 4/6$。
- 分子：$B \cap (A \cup B)$。因為 $B$ 已經被包在聯集裡了，交集就是 $B$ 本身 $\{4, 5, 6\}$，機率 $P(B) = 3/6$。
- **計算：** $P(B \mid A \cup B) = \frac{3/6}{4/6} = \frac{3}{4}$ (0.75)。
- **觀念救援：** 不要被符號嚇到，只要確定分子是「兩者的重疊」，而「自己」跟「聯集」的重疊永遠是「自己」！

### 🎯 弱點打擊 2：貝氏定理與樹狀圖 (Ch 2)

**3. 醫學檢測問題**
- **解題步驟 (畫樹狀圖)：**
  - 第 1 層：患病 (0.01)、未患病 (0.99)。
  - 第 2 層 (患病分支)：陽性 (0.90)、陰性 (0.10)。
  - 第 2 層 (未患病分支)：陽性 (0.05)、陰性 (0.95)。
- **(a) 全機率 (求陽性機率分母)：**
  - $\text{陽性} = (0.01 \times 0.90) + (0.99 \times 0.05) = 0.009 + 0.0495 = 0.0585$。
- **(b) 貝氏定理 (給定陽性，求患病)：**
  - $\frac{\text{患病那條路徑}}{\text{陽性總機率}} = \frac{0.009}{0.0585} \approx 0.1538$ (約 15.38%)。
- **觀念救援：** 你看！即使檢測準確度很高(90%)，陽性後真正患病的機率還是只有 15%，這就是有名的「偽陽性陷阱」。只要樹狀圖有畫出來，就不會算錯。

### 🎯 弱點打擊 3：連續型與條件期望值陷阱 (Ch 4)

**4. 期望值的「少乘一個 x」陷阱**
- **(a) 證明 PDF：**
  - $\int_0^2 \frac{1}{2} x dx = [\frac{1}{4} x^2]_0^2 = \frac{4}{4} - 0 = 1$。(正確)
- **(b) 期望值 $E[X]$：**
  - **關鍵：要多乘一個 $x$！**
  - $E[X] = \int_0^2 x \cdot (\frac{1}{2} x) dx = \int_0^2 \frac{1}{2} x^2 dx = [\frac{1}{6} x^3]_0^2 = \frac{8}{6} = \frac{4}{3}$。

**5. 條件期望值的大魔王**
- **(a) 求分母機率 $P(M)$：**
  - $P(X \ge 1) = \int_1^2 \frac{1}{2} x dx = [\frac{1}{4} x^2]_1^2 = \frac{4}{4} - \frac{1}{4} = \frac{3}{4}$ (0.75)。
- **(b) 條件期望值 $E[X \mid X \ge 1]$：**
  - 分子的積分：$\int_1^2 x \cdot (\frac{1}{2} x) dx = [\frac{1}{6} x^3]_1^2 = \frac{8}{6} - \frac{1}{6} = \frac{7}{6}$。
  - **關鍵陷阱：一定要除以分母機率！**
  - $E[X \mid X \ge 1] = \frac{7/6}{3/4} = \frac{7}{6} \times \frac{4}{3} = \frac{28}{18} = \frac{14}{9}$。

### 🎯 弱點打擊 4：變異數線性組合 (綜合)

**6. 係數平方與常數陷阱**
- **(a) $E[W]$：**
  - $E[4X - 3Y + 5] = 4E[X] - 3E[Y] + 5$
  - $= 4(5) - 3(-1) + 5 = 20 + 3 + 5 = 28$。
- **(b) $Var(W)$：**
  - **陷阱救援：常數 5 消失、係數 4 與 -3 要平方！**
  - $Var(4X - 3Y + 5) = 4^2 Var(X) + (-3)^2 Var(Y)$
  - $= 16(2) + 9(3) = 32 + 27 = 59$。
