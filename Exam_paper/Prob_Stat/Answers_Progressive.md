## 詳細解答與步驟解析

### 一、基礎機率與貝氏定理 (Ch 2)

**1. 獨立與互斥事件**
- **(a) 互斥事件：**
  - **解題步驟：** 互斥代表兩事件不可能同時發生，即 $P(A \cap B) = 0$。
  - **計算：** $P(A \cup B) = P(A) + P(B) - P(A \cap B) = 0.4 + 0.5 - 0 = 0.9$
- **(b) 獨立事件：**
  - **解題步驟：** 獨立代表兩事件發生互不影響，即 $P(A \cap B) = P(A) \times P(B)$。
  - **計算：** 
    - $P(A \cap B) = 0.4 \times 0.5 = 0.2$
    - $P(A \cup B) = 0.4 + 0.5 - 0.2 = 0.7$
- ⚠️ **常見錯誤陷阱：** 常有人將「獨立」與「互斥」搞混。互斥代表交集為0；獨立代表交集為兩者相乘！

**2. 條件機率**
- **解題步驟：** 使用條件機率公式 $P(E \mid F) = \frac{P(E \cap F)}{P(F)}$。
- **計算：** 
  - $P(A \mid A \cup B) = \frac{P(A \cap (A \cup B))}{P(A \cup B)}$
  - 因為 $A$ 是 $A \cup B$ 的子集，所以 $A \cap (A \cup B) = A$。
  - 上方分子為 $P(A) = 0.4$。分母為 $P(A \cup B) = 0.7$ (由1b得知)。
  - 答案： $0.4 / 0.7 = 4/7$。

**3. 貝氏定理與樹狀圖**
- **解題步驟：** 畫樹狀圖。第一層為甲(0.6)、乙(0.4)。第二層為不良品/良品。甲生出不良品為 0.02，乙生出不良品為 0.05。
- **(a) 不良品機率 (全機率定理)：** 
  - $P(\text{不良}) = P(\text{不良} \mid 甲)P(甲) + P(\text{不良} \mid 乙)P(乙)$
  - $= (0.02 \times 0.6) + (0.05 \times 0.4) = 0.012 + 0.020 = 0.032$
- **(b) 給定不良，來自乙的機率 (貝氏定理)：**
  - $P(乙 \mid \text{不良}) = \frac{P(\text{不良} \mid 乙)P(乙)}{P(\text{不良})} = \frac{0.020}{0.032} = \frac{20}{32} = \frac{5}{8}$ = 0.625

### 二、離散型隨機變數 (Ch 3)

**4. PMF 與期望值**
- **(a) 求 c：** 
  - 所有機率總和為 1：$\sum_{x=1}^{4} c \cdot x = 1$
  - $c(1 + 2 + 3 + 4) = 1 \implies 10c = 1 \implies c = 0.1$
- **(b) 期望值 $E[X]$：**
  - $E[X] = \sum x \cdot P(X=x) = \sum x(0.1x) = 0.1 \sum x^2$
  - $= 0.1 \times (1^2 + 2^2 + 3^2 + 4^2) = 0.1 \times (1 + 4 + 9 + 16) = 0.1 \times 30 = 3$

**5. 變異數與函數期望值陷阱**
- **(a) $E[X^2]$：**
  - $E[X^2] = \sum x^2 \cdot P(X=x) = \sum x^2 (0.1x) = 0.1 \sum x^3$
  - $= 0.1 \times (1^3 + 2^3 + 3^3 + 4^3) = 0.1 \times (1 + 8 + 27 + 64) = 0.1 \times 100 = 10$
- **(b) $Var(X)$：**
  - $Var(X) = E[X^2] - (E[X])^2 = 10 - 3^2 = 10 - 9 = 1$
- **(c) $Var(X^2)$：**
  - **核心公式：** $Var(Y) = E[Y^2] - (E[Y])^2$。令 $Y = X^2$，則 $Var(X^2) = E[X^4] - (E[X^2])^2$。
  - 先算 $E[X^4] = \sum x^4(0.1x) = 0.1 \sum x^5 = 0.1 \times (1 + 32 + 243 + 1024) = 0.1 \times 1300 = 130$
  - $Var(X^2) = 130 - (10)^2 = 130 - 100 = 30$
  - ⚠️ **常見錯誤陷阱：** 以為 $Var(X^2) = (Var(X))^2$，這是絕對錯誤的！歷屆考題考過，請特別注意。

### 三、聯合機率分配 (補充)

**6. 超幾何分配與聯合 PMF**
- **(a) $P(X=1, Y=1)$：**
  - 總球數 = $3 + 2 + 1 = 6$。抽出 2 顆，總組合數 $C^6_2 = 15$。
  - 1紅 1藍 的組合數 = $C^3_1 \times C^2_1 = 3 \times 2 = 6$。
  - 機率 = $6 / 15 = 2 / 5 = 0.4$。
- **(b) $X$ 的邊際機率與 $E[X]$：**
  - $X$ 可能的值為 0, 1, 2。
  - $P(X=0) = C^3_0 \times C^3_2 / 15 = 1 \times 3 / 15 = 3/15$
  - $P(X=1) = C^3_1 \times C^3_1 / 15 = 3 \times 3 / 15 = 9/15$
  - $P(X=2) = C^3_2 \times C^3_0 / 15 = 3 \times 1 / 15 = 3/15$
  - 檢查：$3/15 + 9/15 + 3/15 = 1$ (正確)。
  - $E[X] = 0(3/15) + 1(9/15) + 2(3/15) = 15/15 = 1$。

### 四、連續型隨機變數 (Ch 4)

**7. PDF 性質與常數求解**
- **解題步驟：** $\int f(x) dx = 1$
- **計算：** $\int_0^2 k x^2 dx = k [\frac{x^3}{3}]_0^2 = k \frac{8}{3} = 1 \implies k = \frac{3}{8}$
- 得到 $f_X(x) = \frac{3}{8}x^2$ ($0 < x \le 2$)

**8. 連續型期望值與變異數**
- **(a) $E[X]$：** $\int_0^2 x \cdot (\frac{3}{8}x^2) dx = \frac{3}{8} \int_0^2 x^3 dx = \frac{3}{8} [\frac{x^4}{4}]_0^2 = \frac{3}{8} \cdot \frac{16}{4} = \frac{3}{8} \cdot 4 = \frac{3}{2} = 1.5$
- **(b) $Var(X)$：**
  - 先求 $E[X^2] = \int_0^2 x^2 (\frac{3}{8}x^2) dx = \frac{3}{8} \int_0^2 x^4 dx = \frac{3}{8} [\frac{x^5}{5}]_0^2 = \frac{3}{8} \cdot \frac{32}{5} = \frac{12}{5} = 2.4$
  - $Var(X) = E[X^2] - (E[X])^2 = 2.4 - (1.5)^2 = 2.4 - 2.25 = 0.15$ (或 $\frac{3}{20}$)

**9. 條件期望值**
- **(a) $P(A) = P(X \le 1)$：**
  - $\int_0^1 \frac{3}{8}x^2 dx = \frac{3}{8} [\frac{x^3}{3}]_0^1 = \frac{1}{8} = 0.125$
- **(b) 條件期望值 $E[X \mid A]$：**
  - **核心公式：** $E[X \mid X \le 1] = \frac{1}{P(X \le 1)} \int_0^1 x \cdot f_X(x) dx$
  - 分子的積分：$\int_0^1 x (\frac{3}{8}x^2) dx = \frac{3}{8} \int_0^1 x^3 dx = \frac{3}{8} [\frac{x^4}{4}]_0^1 = \frac{3}{32}$
  - $E[X \mid A] = \frac{3/32}{1/8} = \frac{3}{32} \times 8 = \frac{3}{4} = 0.75$
  - ⚠️ **常見錯誤陷阱：** 忘記除以 $P(A)$ (即分母的 $1/8$)！

**10. 線性組合與綜合應用**
- **解題步驟：** $E[aX + bY + c] = aE[X] + bE[Y] + c$；獨立時， $Var(aX + bY + c) = a^2 Var(X) + b^2 Var(Y)$。
- **計算 $E[W]$：** $E[3X - 2Y + 1] = 3E[X] - 2E[Y] + 1 = 3(1.5) - 2(2) + 1 = 4.5 - 4 + 1 = 1.5$
- **計算 $Var(W)$：** $Var(3X - 2Y + 1) = 3^2 Var(X) + (-2)^2 Var(Y) = 9(0.15) + 4(1) = 1.35 + 4 = 5.35$ (常數項變異數為0)
