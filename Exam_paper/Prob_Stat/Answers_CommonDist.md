## 詳細解答與步驟解析

### 🎯 補漏題型 1：聯合機率分配的表格法

**1. 聯合與邊際機率**
- **(a) X 的邊際機率：**
  - **解題步驟：** 把 $X$ 所在的「一整列 (Row)」機率全部加起來。
  - $P(X=1) = 0.1 + 0.2 + 0.1 = 0.4$
  - $P(X=2) = 0.2 + 0.1 + 0.3 = 0.6$
  - *(檢查：$0.4 + 0.6 = 1$，正確)*
- **(b) Y 的邊際機率 $P(Y=2)$：**
  - **解題步驟：** 把 $Y=2$ 所在的「一整欄 (Column)」機率全部加起來。
  - $P(Y=2) = 0.2 + 0.1 = 0.3$
- **(c) X 與 Y 是否獨立？**
  - **解題步驟：** 若獨立，必須對於「每一個格子」都滿足 $P(X=x, Y=y) = P(X=x) \times P(Y=y)$。只要找到一個格子不符合，就是不獨立。
  - 我們來檢查 $X=1, Y=2$ 這個格子：
  - 查表得知：$P(X=1, Y=2) = 0.2$
  - 計算邊際相乘：$P(X=1) \times P(Y=2) = 0.4 \times 0.3 = 0.12$
  - 因為 $0.2 \neq 0.12$，所以 $X$ 與 $Y$ **不獨立 (Not Independent)**。

### 🎯 補漏題型 2：測量誤差

**2. Bias 與 Random Error**
- **(a) 系統誤差還是隨機誤差？**
  - **解答：** 屬於 **Bias (系統誤差)**。
  - **解析：** 根據講義定義，Bias 是一種「每次量測都相同」的誤差 (is error that is the same for every measurement)。而 Random Error 則是每次量測都會變動、長遠來看會互相抵銷平均為 0 的誤差。既然是固定少 2 克，那就是 Bias。
- **(b) 量測值的期望值：**
  - 公式：$E[\text{Measured}] = E[\text{True}] + E[\text{Bias}] + E[\text{Random}]$
  - 帶入數值：$E[\text{Measured}] = 100 + (-2) + 0 = 98$ 克。
  - **解析：** 因為真實數值和 Bias 都是常數，而 Random Error 的期望值為 0，所以最終量測出的期望值就是真實重量減去 Bias。

### 🎯 補漏題型 3：常態分配法則

**3. 68-95-99.7% 法則 (Empirical Rule)**
- 題目給定：$\mu = 50$, $\sigma = 2$。
- $\mu \pm 1\sigma$ 的範圍：$50 \pm 2 \implies [48, 52]$
- $\mu \pm 2\sigma$ 的範圍：$50 \pm 4 \implies [46, 54]$
- $\mu \pm 3\sigma$ 的範圍：$50 \pm 6 \implies [44, 56]$

- **(a) 48 毫米 到 52 毫米 之間：**
  - 這是落在 $\mu \pm 1\sigma$ 的區間。
  - **解答：** 大約有 **68%**。
- **(b) 46 毫米 到 54 毫米 之間：**
  - 這是落在 $\mu \pm 2\sigma$ 的區間。
  - **解答：** 大約有 **95%**。
- **(c) 大於 54 毫米：**
  - **解題步驟：** $\mu \pm 2\sigma$ ($46 \sim 54$) 佔了 95%。這代表剩下的 5% 是落在這區間之外（也就是 $<46$ 和 $>54$ 的部分）。
  - 因為常態分配是「左右對稱 (Symmetric)」的，所以 $>54$ 的機率與 $<46$ 的機率一樣大。
  - 因此，大於 54 毫米的機率是剩下 5% 的一半。
  - **解答：** $5\% \div 2 = \mathbf{2.5\%}$。
  - *(這題是歷屆非常愛考的尾部機率觀念！)*
